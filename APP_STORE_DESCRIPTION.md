# OpenPaste - App Store 推荐文案

## 应用简介（副标题）
**开源剪贴板历史工具，让每一次复制都有迹可循**

## 应用描述

### 核心功能

**📋 智能剪贴板历史**
- 自动保存所有复制内容，支持文本、图片、链接、文件等多种格式
- 完整保留原始剪贴板数据，确保粘贴时格式不丢失
- Core Data 持久化存储，重启后数据依然保留

**⌨️ 全局快捷键**
- 默认快捷键 `⌘⇧V`，一键唤起历史面板
- 可自定义快捷键，适配个人使用习惯
- 状态栏图标常驻，点击即可快速访问

**🎯 智能分类**
- **最近**：所有内容按时间排序
- **文本**：纯文本和富文本内容
- **代码**：识别多种编程语言代码片段
- **图片**：PNG、JPG、GIF 等图片格式
- **文件**：文件和文件夹引用
- **链接**：网页 URL 链接
- **邮箱**：电子邮件地址
- **电话**：电话号码
- **颜色**：十六进制颜色代码
- **收藏**：4 个自定义收藏夹，用不同颜色标记重要内容

**🔍 强大搜索**
- 实时搜索历史记录
- 按内容类型、来源应用、日期范围筛选
- 智能匹配，快速定位所需内容

**⚙️ 灵活设置**
- 自定义数据保留期限（1-90 天）
- 设置历史记录数量上限
- 一键清理过期数据
- 自定义快捷键

**🎨 精美设计**
- 原生 macOS 风格，毛玻璃效果
- 流畅的侧边滑入动画
- 深色/浅色模式自动适配
- 紧凑侧边栏 + 宽敞内容区布局

### 为什么选择 OpenPaste？

**开源透明**
- 完全开源，代码可审查
- 无隐私隐患，所有数据仅存储在本地

**性能优异**
- 轻量级设计，占用资源极少
- 响应迅速，即时检索

**隐私安全**
- 数据不上传云端，完全本地存储
- App Sandbox 沙盒隔离，保护系统安全
- 可设置自动清理敏感数据

### 使用场景

- **开发者**：快速复用代码片段
- **设计师**：管理颜色代码和素材
- **文案编辑**：在多个版本间切换文案
- **数据录入**：批量复制粘贴信息
- **日常办公**：提升工作效率，告别重复复制

### 技术亮点

- SwiftUI + Core Data 双引擎驱动
- 完整的剪贴板格式保留机制
- 响应式状态管理，UI 实时更新
- macOS 14.0+ 原生适配
