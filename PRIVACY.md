# OpenPaste 隐私政策 / Privacy Policy

**最后更新：2026年3月31日**
**Last Updated: March 31, 2026**

## 中文 / Chinese

### 1. 数据收集

OpenPaste **不收集**任何用户数据。

- ❌ 不收集个人信息
- ❌ 不收集使用行为数据
- ❌ 不收集设备信息
- ❌ 不收集位置信息
- ❌ 不使用分析工具
- ❌ 不使用第三方服务

### 2. 数据存储

OpenPaste 所有数据均**存储在您的设备本地**：

- ✅ 剪贴板历史记录存储在本地 Core Data 数据库
- ✅ 图片文件存储在本地应用沙盒目录
- ✅ 应用设置存储在本地 UserDefaults

### 3. 数据访问

OpenPaste 启用了 **App Sandbox** 隔离机制：

- 应用仅在用户主动操作时访问剪贴板
- 应用无法访问其他应用的数据
- 应用不会将数据发送到任何服务器

### 4. 数据删除

用户可以随时删除所有数据：

- 在应用设置中点击"清除所有数据"
- 卸载应用将删除所有本地数据

### 5. 第三方服务

OpenPaste **不使用任何第三方服务**。

### 6. 开源透明

OpenPaste 是开源软件，源代码公开在 GitHub：

- https://github.com/fictionking/OpenPaste

您可以自行审查代码，验证我们的隐私承诺。

### 7. 联系方式

如有隐私相关问题，请通过以下方式联系：

- GitHub Issues: https://github.com/fictionking/OpenPaste/issues
- Email: (可选填)

## English

### 1. Data Collection

OpenPaste **does not collect** any user data.

- ❌ No personal information collected
- ❌ No usage analytics
- ❌ No device information collected
- ❌ No location data
- ❌ No third-party analytics or services

### 2. Data Storage

All OpenPaste data is **stored locally on your device**:

- ✅ Clipboard history stored in local Core Data database
- ✅ Images stored in local app sandbox directory
- ✅ Settings stored in local UserDefaults

### 3. Data Access

OpenPaste uses **App Sandbox** isolation:

- App only accesses clipboard when you actively use it
- App cannot access data from other apps
- App never sends data to any server

### 4. Data Deletion

You can delete all data at any time:

- Click "Clear All Data" in app settings
- Uninstalling the app removes all local data

### 5. Third-Party Services

OpenPaste **does not use any third-party services**.

### 6. Open Source & Transparent

OpenPaste is open source software. Source code is available on GitHub:

- https://github.com/fictionking/OpenPaste

You can review the code to verify our privacy commitments.

### 7. Contact

For privacy-related questions, please contact:

- GitHub Issues: https://github.com/fictionking/OpenPaste/issues
- Email: (optional)
